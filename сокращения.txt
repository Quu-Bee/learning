F12 - перейти к переменной
F2 - переименовать переменную
TAB/shift + TAB - сместить табуляцию влево/вправо
Shift_Del - удалить строку
Shift + Alt + ↓/↑ — дублирование строки с курсором вниз
Alt + ↑/↓ - перемещение строки с курсором вверх или вниз
ctrl + / - закоментировать строку
ctrl + D - выделить слово с курсором
ctrl + D + D... - выделить следующее идентичное слово 
ctrl + L - выделить всю строку
ctrl + S - отформатировать документ
ctrl + k, ctrl + c - закомментить 
ctrl + k, ctrl + u - раскомментить
